Find the title of each film
```SQL
SELECT Title 
FROM movies;
```

Find the director of each film
```SQL
SELECT Director 
FROM movies;
```

Find the title and director of each film
```SQL
SELECT Title, Director 
FROM movies;
```

Find the title and year of each film
```SQL
SELECT Title, Year
FROM movies;
```

Find all the information about each film
```SQL
SELECT *
FROM movies;
```
